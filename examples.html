jQuery.showWhen examples

Basic usage:

    $('#example1').showWhen($('#example1-toggle'));

#example1

Using callbacks to customize

    $('#example2').showWhen('#example2-toggle',
        null,
        // hideCallback
        function($target){
            $target.fadeOut().fadeIn().faceOut();
        },
        // showCallback
        function($target){
            $target.fadeIn().fadeOut().fadeIn();
        });
